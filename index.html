Hello There

CodeArena is an online competitive coding platform which enables you to practice and compete in 54 different languages.

Why?

CodeArena is an awesome solution to host institutional coding contest and practice online. This can boost the student's moral and interest in coding.

Use the Space key to navigate through all slides.


How?

CodeArena is a complete fullstack project which uses cutting edge technology and mordern application architecture to solve various typical problems like compiling and managing data.

When?

We came up with this idea after observing the lack of interest for competitvie coding in Truba.


TechStack

We use python and javascript as our primary programming languages and Linux NginX and Gunicorn as our server infrastructure.

Django

Django is an awesome backend web framework for perfectionists with deadlines. Its MVC pattern is the most stable and talk of the town design pattern due to current developement in microservices architecture. It's easy to use documentation and stable codebase helped us to create this impeccable application.

Angular & Bootstrap

Angular is a platform that makes it easy to build applications with the web. Angular empowers developers to build applications that live on the web, mobile, or the desktop. Bootstrap is used for styling the reusable components generated with angular.

NginX and Gunicorn

Gunicorn is static file server which works in front of NginX. Proper Nginx configuration is the key reason which boosts up the speed of our application.
